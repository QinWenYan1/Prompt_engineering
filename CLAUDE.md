## Repository Purpose

This repository organizes materials for "Prompting For Everyone" and "Anthropic's Prompt Engineering Tutorial" — educational content about prompt engineering.

## Project Structure

- This is a **documentation/content** repository. There is no build system, test suite, or linting.
- Primary language for content: Chinese (简体中文), based on the README.
- Licensed under MIT.

## Content Guidelines

- Content should be organized in a logical, browsable directory structure as it grows.
- Keep example prompts self-contained and well-commented so readers can understand them in isolation.
